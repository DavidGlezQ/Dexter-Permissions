# Permisos-Con-Dexter
 App para usar distintos permisos con la libreria Dexter.
 Funciones principales:
 
-UI Dark: para esta app se aplica un modo oscuro y aplicando diferentes diseños y colores a botones como modo de prueba.

-Boton de Cámara: Para este boton se utliliza Composite Dexter, el cual hace que al denegar el permiso ya sea permanently o denied manda un cuadro de dialogo al usuario indicando que necesita aceptar el permiso para poder tomar fotos.
Boton de Contactos: Para este boton se utliza Composite Dexter pero esta vez con un SnackBar en el cual al denegar el permiso de cualquier manera muestra el SnackBar con un boton donde puede ir a la configuracion de la app para cambiarlo desde ahí.

-Boton de Audio: Para este boton solo se utiliza Dexter para aceptar o denegar el permiso pero sin mostrar ningun mensaje al ususario.

-Boton para aceptar todos los permisos: Para este boton se asigna una funcion donde se pueden aceptar todos los permisos y los va mostrando en orden, no muestra ningun mensaje al usuario.
